# Everything but the kitchen sink in one static website

This is a test of my skills implementing Astro/Bootstrap with light mode, dark mode, translations for English, German, and Arabic (including support for RTL styling), video support, responsive image sizes, GSAP animations (fade transitions, parallax scroll, number counting up).
